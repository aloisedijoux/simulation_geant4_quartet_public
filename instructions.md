
1. Define the geometry in `setup_config_linux.g4bl` (refer to the manual: [https://www.muonsinc.com/Website1/Muons/G4beamlineUsersGuide.pdf](https://www.muonsinc.com/Website1/Muons/G4beamlineUsersGuide.pdf))

2. Run `scan_optimize_290226.py` and configure the paths inside:

```python
G4BL      = "/home/aloiselkb/Downloads/G4beamline-3.08/bin/g4bl"  # path to the directory where g4bl was installed
G4BL_FILE = "/home/aloiselkb/g4bl_simu/setup_config_linux.g4bl"  # path to the directory where the setup_config_linux.g4bl file is located

BASE_OUT = Path("/home/aloiselkb/g4bl_simu/test22042026/")  # output path
```

3. Run `analyse_tous_conf.py` to analyze simulation results:
It goes through all the run folders in a root directory and for each, reconstructs the muon trajectory through the gas cell by reading the `ScanZ_.txt` files generated by the simulation. It then generates a series of figures: the particle count distribution per z-plane, the muon stopping profile (where they stop in the gas), and the linear energy loss −dE/dx as a function of z. All results are saved in a directory `output_<config_name>/` specific to each configuration, with a summary `log.txt` file. Errors from individual runs are captured and reported at the end without interrupting the processing of other configurations.

4. Run `gaussiennes.py`. Configure:

 `RACINE`: path to the folder containing the runs
 `FILTRE`: substring to only process a subset of the runs (None = all)
 `L_GAZ`, `N_SCAN`: geometry (must match the simulation)

Outputs — in each `output_<config_name>/`:

 `fig_fit_gaussienne.png`: stop distribution + Gaussian fit curve (or annotation "no significant stops" if the muons pass through)
 `log_fit_gaussienne.txt`: values of mu, sigma, FWHM with uncertainty

5. Run `grouper_gaussiennes.py`:
Configure:

 `BASE_DIR`: directory containing the subdirectories `output_` generated by `gaussiennes.py`
 `MATERIAU`: filter by window material (e.g., "Ti", "Al", "KAPTON")

Output: A single file in `BASE_DIR`:

 `all_gaussiennes_<material>.png`: grid of all `fig_fit_gaussienne.png` figures, automatically arranged with the most varying parameter in rows, the second one in columns, and the fixed parameters in the title.
