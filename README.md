# simulation_geant4_quartet_public
Here is the GitHub repository containing simulations of a particle beam (here, muons) of a certain energy passing through a geometry constructed in G4 beamline, with analysis scripts such as histograms of stopping positions. This is useful for optimizing parameters such as the pressure to apply in a cell to stop particles, the initial energy, the input window (material and geometry), and the geometry of the cell. Everything can be configured and modified in the .g4bl file, but it is important to remember to set the parameters to "-unset" because here, the muon beam is from the piE1 line at PSI.
